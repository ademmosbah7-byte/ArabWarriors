# Arab Warriors

Welcome to the Arab Warriors faction! Experience legendary Middle Eastern warriors in TABS!

### Join the [TABS Mod Center](https://discord.gg/zrs44qyp7S) for support and discussions!

## Installation

1. Install [BepInEx](https://thunderstore.io/c/totally-accurate-battle-simulator/p/BepInEx/BepInExPack_TABS/) (v5.4.16+)
2. Download Arab Warriors mod
3. Extract to `TABS/BepInEx/plugins/`
4. Launch TABS and enjoy!

## Troubleshooting

**Mod doesn't appear?**
- Ensure BepInEx is installed correctly
- Check `BepInEx/plugins/ArabWarriors/` folder structure
- Review `BepInEx/LogOutput.log`

**Units not working?**
- Verify mod version compatibility
- Update BepInEx to v5.4.16+
- Remove conflicting mods

**Need help?**
- Join [TABS Mod Center Discord](https://discord.gg/zrs44qyp7S)
- Check BepInEx logs
- Report issues with detailed info
